Hexo + Gitpages完成个人博客搭建

1.准备工作

  • github账号
  • 新建一个与github账号同名的库 账号id.github.io
  • node.js环境
  • 安装git

2.下载Hexo

npm install -g hexo-cli

3.建站

$ hexo init <folder>

$ cd <folder>

$ npm install

4.配置

4.1配置_config.xml

参考：https://hexo.io/zh-cn/docs/configration.html

4.2配置github ssh

先安装hexo-deployer-git插件

$ npm install hexo-deployer-git -save

修改hexo的配置文件

4.3theme

目录位置：主目录>themes theme

执行代码$ git clone <theme github地址>

一般主题的about页面需要自己新建

5.主要命令

主目录初始化

$ hexo init <folder>

新建文章(layout可以省略)

$ hexo new [layout] <title>

文章push到github上命令

$ hexo clean
$ hexo g
$ hexo d

6.本地测试

$ hexo clean 清除缓存
$ hexo g 挂载
$ hexo s 启动服务

执行完成之后访问 localhost:4000

注：匆匆记录一下过程，帮助意义不大。如需建站，谨慎使用。
